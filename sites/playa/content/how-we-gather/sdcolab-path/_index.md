1. Check availability in the tool library system
2. Reserve if needed
3. Inspect the tool at checkout (make sure it's not already broken)

5. Report any damage immediately

3. Demonstrate safe operation
4. Pass the quiz

**Be patient with yourself.** Learning tools takes time. That's okay.

## Safety Rules â€” The Pack Protects the Pack ğŸ¾

These aren't bureaucracy. They're how we take care of each other.

### Universal Rules (Always Apply)

- **No working alone on power tools.** Ever. Find a buddy.
- **Closed-toe shoes required.** No flip-flops in the shop.
- **Safety glasses in shop areas.** Protect your eyeballs.
- **No alcohol/drugs before or during work.** This should be obvious.
- **Clean as you go.** Don't leave messes for the next person.

### Tool-Specific Safety

- Follow posted procedures for each tool
- Use required PPE (posted at each station)
- Don't use tools you're not certified for
- **When in doubt, ask.** Seriously.

### When Things Go Wrong

- **All injuries get reported immediately.** No exceptions.
- **Near-misses get reported too.** They help us prevent real injuries.
- **No blame culture.** We learn from incidents.
- **First aid kit locations are posted.** Know where they are.
