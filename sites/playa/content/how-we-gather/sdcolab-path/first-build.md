## What If I Mess Up?

You will. Everyone does. That's how learning works.

**If you break a tool:**
- Stop
- Tell someone immediately
- Don't try to hide it

**No blame culture.** We'd rather know about a broken tool than have the next person get hurt.

**If your project fails:**
- Learn from it
- Try again (or try something else)
- Talk about what went wrong (others can learn too)

Failure is just research. ğŸ”¬
