## Areas

- **Main Workshop** — 2,500 sq ft: welding, metal shop, wood shop, assembly
- **Electronics Lab** — Soldering, oscilloscopes, 3D printers, laser cutter
- **Art Space** — Painting, fabric/sewing, storage
- **Outdoor Yard** — Large fabrication, fire testing
- **Common Areas** — Kitchen, bathroom, library

## Tool Access Levels

| Level | Access | How to Get |
|-------|--------|------------|
| Basic | Hand tools, assembly | Complete orientation |
| Intermediate | Power tools, electronics | Take class |
| Advanced | Welding, laser, heavy equipment | Class + supervised hours |

## Rules

1. Clean up after yourself
2. Put tools back where they belong
3. Ask before using unfamiliar tools
4. Report damage immediately
5. No working alone on dangerous tools
6. No drugs/alcohol
